# Antigravity Kit (Web)

> Web geliştirme için YZ Ajan Şablonları

## Kurulum

Proje kök dizininde (bu paketin `.agent` olarak eklendiğini varsayarsak):

```bash
# Doğrudan Klonlama
git clone https://github.com/erenmustafaozdal/antigravity-kit-tr .agent
```

## İçerik

- **Frontend Ajanları**: React, Next.js, Vue, Tailwind uzmanları
- **Backend Ajanları**: Node.js, API, Veritabanı uzmanları
- **Web İş Akışları**: Hızlı geliştirme için slash komutları

Daha fazla bilgi için ana `README.md` dosyasına bakın.
